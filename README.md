PostVista is a clean and responsive React application that fetches blog posts from a public REST API and displays them in a modern card-based layout. The project demonstrates core React concepts such as Hooks, API integration, conditional rendering, and responsive UI design.

🚀 Features

Fetches real-time data from a public REST API

Displays content in a responsive grid layout

Handles Loading and Error states gracefully

Clean and minimal UI

Fully responsive across devices

Built using functional components and React Hooks

🛠️ Technologies Used

React.js

JavaScript (ES6+)

CSS3 (Responsive Grid)

Fetch API

JSONPlaceholder Public API
https://jsonplaceholder.typicode.com/

⚙️ Installation & Setup

Clone the repository

Run npm install

Start the development server:

npm start

The app will run on http://localhost:3000
